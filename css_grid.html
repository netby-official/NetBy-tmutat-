CSS Grid használata

A CSS Grid egy rendkívül erőteljes eszköz, amely lehetővé teszi komplex weboldalak elrendezésének létrehozását. Az alábbiakban bemutatjuk, hogyan használhatod a CSS Grid-et:

1. lépés: Grid konténer létrehozása

A Grid rendszerhez először egy konténer elemet kell létrehoznod, amely az display: grid; beállítást használja:

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

Ez a kód három egyenlő oszlopot hoz létre, és 10px-es rést ad az elemek között.

2. lépés: Grid elemek hozzáadása

A Grid konténerben elhelyezett elemek automatikusan grid elemek lesznek. Az elemeket elhelyezheted a grid rendszerben az alábbi módon:

.grid-item {
    background-color: lightgray;
    padding: 20px;
    text-align: center;
}

Ez a kód formázza a grid elemeket, így azok háttérszíne világosszürke lesz, középre igazított szöveggel.

3. lépés: Grid vonalak használata

Beállíthatod, hogy a grid elemek több sorra és oszlopra terjedjenek ki:

.item1 {
    grid-column: span 2;
    grid-row: span 2;
}

Ez a kód azt jelenti, hogy az .item1 osztályú elem két oszlopra és két sorra terjed ki.

4. lépés: Grid területek használata

Grid területeket is definiálhatsz, hogy könnyebben kezelhesd az elrendezést:

.grid-container {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
}

.header {
    grid-area: header;
}

.sidebar {
    grid-area: sidebar;
}

.main {
    grid-area: main;
}

.footer {
    grid-area: footer;
}

Ez a kód egy olyan elrendezést hoz létre, ahol a fejléc az egész szélességben terjed, a bal oldali sáv és a fő tartalom külön helyet kapnak, és a lábléc szintén az egész szélességben terjed.
