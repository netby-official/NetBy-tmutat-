JavaScript alapok

JavaScript egy dinamikus programozási nyelv, amelyet a weboldalak interaktivitásának növelésére használnak. Az alábbiakban bemutatjuk a JavaScript alapjait:

1. lépés: JavaScript beillesztése HTML-be

JavaScript kódot beilleszthetsz egy HTML fájlba a <script> elem használatával:

<script src="script.js"></script>

Vagy közvetlenül beágyazhatod a kódot:

<script>
    alert('Helló világ!');
</script>

2. lépés: Alapvető JavaScript szintaxis

Íme néhány alapvető JavaScript szintaktikai elem:

  • Változók: let x = 10;
  • Függvények: function greet() { alert('Helló!'); }
  • Objektumok: let person = { name: 'John', age: 30 };

3. lépés: DOM manipuláció

A Document Object Model (DOM) segítségével a JavaScript programozók módosíthatják a HTML elemeket:

document.getElementById('myElement').innerText = 'Új szöveg';

4. lépés: Eseménykezelés

Az eseménykezelés lehetővé teszi, hogy reakciókat hozz létre felhasználói interakciókra:

document.getElementById('myButton').addEventListener('click', function() {
    alert('Gombra kattintottál!');
});
