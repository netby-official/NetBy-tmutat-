Képek optimalizálása

A weboldalak gyorsabb betöltése érdekében fontos, hogy a képeket optimalizáljuk. Ebben az útmutatóban megtanulhatod, hogyan teheted ezt meg.

1. lépés: Megfelelő képformátum kiválasztása

A különböző képformátumok eltérő előnyökkel járnak:

  • JPEG: Ideális fotókhoz, mivel tömöríthető és kis fájlméretet eredményez, de némi minőségromlással.
  • PNG: Ideális grafikonokhoz, logókhoz, ahol a minőség megőrzése fontos, átlátszó háttérrel is használható.
  • WebP: Egy újabb formátum, amely kis fájlméretet biztosít, megőrizve a minőséget.

2. lépés: Képek tömörítése

A képek tömörítése fontos a fájlméret csökkentéséhez. Használj online eszközöket, mint például a TinyPNG vagy CompressJPEG a képek tömörítésére. Ezek az eszközök jelentősen csökkenthetik a fájlméretet anélkül, hogy a minőség jelentősen romlana.

3. lépés: Képméret csökkentése

Győződj meg róla, hogy a képek mérete (pixelben) megfelel a weboldalon való megjelenítéshez. Például, ha egy kép 4000x3000 pixel méretű, de csak 800x600 pixel méretben jelenik meg az oldalon, akkor érdemes a kép méretét lecsökkenteni erre a méretre.

3. lépés: Képméret csökkentése (folytatás)

Használhatsz képszerkesztő programokat, mint például a Photoshop vagy a GIMP, hogy átméretezd a képeket. Alternatív megoldásként használhatsz online eszközöket, például a ResizeImage.net oldalt.

4. lépés: Lazy loading alkalmazása

A "lazy loading" technika lehetővé teszi, hogy a képek csak akkor töltődjenek be, amikor a felhasználó ténylegesen eléri őket a görgetés során. Ez különösen hasznos hosszú oldalak esetén, ahol sok kép található. Ehhez adj hozzá egy loading="lazy" attribútumot a képekhez:

<img src="example.jpg" alt="Példa kép" loading="lazy">

Ezzel az egyszerű módszerrel csökkentheted a weboldal kezdeti betöltési idejét.

5. lépés: Képek optimalizálása CSS sprite-okkal

A CSS sprite egy technika, amely egyesíti több kis kép (ikon, gomb, stb.) egy nagyobb képbe. Ezzel csökkenthető a HTTP kérések száma, ami gyorsítja az oldal betöltését. A sprite-ok használatához:

  • Hozz létre egy nagy képet, amely tartalmazza az összes kisebb képet.
  • Használd a background-position CSS tulajdonságot, hogy megjelenítsd az egyes képeket a nagy képből:
.icon {
    width: 50px;
    height: 50px;
    background-image: url('sprite.png');
}

.icon-home {
    background-position: 0 0;
}

.icon-search {
    background-position: -50px 0;
}

Ebben a példában egy "sprite.png" nevű képet használunk, amely tartalmazza a különböző ikonokat. Az icon-home és icon-search osztályok az adott ikonok pozícióját határozzák meg.

6. lépés: Képek megfelelő metaadatainak megadása

Ne felejtsd el, hogy a képekhez megadd a megfelelő alt attribútumot, amely leírja a kép tartalmát. Ez segíti a keresőmotorokat és a látássérült felhasználókat, hogy jobban megértsék a kép jelentését:

<img src="example.jpg" alt="Példa kép leírása">

A megfelelő alt attribútum használata javíthatja az oldal SEO-ját és felhasználói élményét.
