Reszponzív webdesign

A reszponzív webdesign lényege, hogy a weboldalak különböző eszközökön, például mobiltelefonokon, táblagépeken és asztali számítógépeken is megfelelően jelenjenek meg.

1. lépés: Reszponzív grid használata

Használj reszponzív grid rendszert, például a CSS Grid-et vagy Flexbox-ot, hogy a tartalom rugalmasan igazodjon a különböző képernyőméretekhez.

2. lépés: Média lekérdezések (Media Queries) alkalmazása

A média lekérdezések segítségével különböző stílusokat alkalmazhatsz a különböző képernyőméretekre:

@media (max-width: 600px) {
    .container {
        width: 100%;
        padding: 10px;
    }
}

3. lépés: Rugalmas képek és videók

Biztosítsd, hogy a képek és videók mérete rugalmasan igazodjon a kijelzőhöz:

img, video {
    max-width: 100%;
    height: auto;
}
