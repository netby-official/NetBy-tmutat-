Weboldal készítése

Ez az útmutató részletesen bemutatja, hogyan hozhatsz létre egy egyszerű weboldalt, még akkor is, ha sosem csináltál ilyet korábban.

1. lépés: HTML fájl létrehozása

A HTML (HyperText Markup Language) a weboldalak alapvető építőköve. Egy egyszerű HTML fájl létrehozásához:

  • Nyiss meg egy szövegszerkesztőt (például a Notepad++ vagy a Visual Studio Code programot).
  • Hozz létre egy új fájlt, és nevezd el index.html-nek.
  • Mentsd el a fájlt a számítógépeden egy új mappában, például „weboldalam” néven.

2. lépés: Alapvető HTML struktúra kialakítása

A következő lépés a HTML fájl struktúrájának megadása. Ez határozza meg, hogy a böngésző hogyan értelmezze a tartalmat. A következő kódot írd be az index.html fájlba:

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Weboldalam</title>
</head>
<body>
    <h1>Üdvözöllek a weboldalamon!</h1>
    <p>Ez egy egyszerű HTML weboldal.</p>
</body>
</html>

A fenti kódban a következők találhatók:

  • <!DOCTYPE html>: Azt jelzi a böngészőnek, hogy HTML5 dokumentummal van dolga.
  • <html lang="hu">: A dokumentum nyelve magyar.
  • <head>: Itt találhatóak a metaadatok, például a karakterkódolás és az oldal címe.
  • <body>: Az oldal tartalma, amely megjelenik a böngészőben.

3. lépés: CSS hozzáadása a megjelenéshez

A CSS (Cascading Style Sheets) határozza meg a weboldal kinézetét. Létrehozhatsz egy külön CSS fájlt, hogy az oldal kinézetét formázd:

  • Hozz létre egy új fájlt a mappádban, és nevezd el styles.css-nek.
  • Írd be a következő kódot a styles.css fájlba:
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    margin: 0;
    padding: 0;
}

.container {
    background-color: #ffffff;
    padding: 20px;
    margin: 20px auto;
    max-width: 800px;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
}

Ez a CSS egyszerűen megadja, hogy az oldal fehér hátterű legyen, középre igazított tartalommal, és szép, tiszta kinézettel.

Most térj vissza az index.html fájlba, és a <head> szekcióban add hozzá a következő sort:

<link rel="stylesheet" href="styles.css">

Ez a link összekapcsolja a HTML fájlt a CSS fájloddal.

4. lépés: Oldal tesztelése és finomítása

Nyisd meg az index.html fájlt egy böngészőben (például a Chrome vagy Firefox használatával), hogy megnézd, hogyan néz ki az oldalad. Ha mindent helyesen csináltál, egy egyszerű, szöveges weboldalt kell látnod, amely a CSS segítségével szépen formázva jelenik meg.

Kísérletezhetsz további HTML elemek (például képek, linkek, listák) hozzáadásával, hogy jobban megértsd, hogyan működnek a weboldalak.
